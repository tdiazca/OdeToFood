OdeToFood MVCOdeToFood - ASP.NET Web (MVC) application

The application has been developed in C# coding language using the MVC 5 framework (.NET framework) and the Entity Framework on MVS 2019. It has been built using the Model-View-Controller arquitecture.

OdeToFood is an application to manage restaurants: allows the user to view and edit restaurants.

Application developed following the 'ASP.NET MVC 5 Fundamentals' course by Scott Allen on Pluralsight.
